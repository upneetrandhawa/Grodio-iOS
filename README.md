# **GRODIO**

##### *A private group radio hosted by users, where they upload an audio file, and it gets played across devices in sync.*
&nbsp;

----

## Features

* Native iOS/iPad/Mac App
* Join/Create Groups
* Login/Create Accounts
* Real-time chat
* Audio player controls.

----
## Tech

* Swift
* Firestore
* Firebase Cloud Storage
